# Arduino_USB_Host

Pinout  
|PIN      | Arduino DUE        | Mini USB Host Shield | ESP32 |
|:------- | :----------------: | :------------------: | ----: |
| Reset   | D7                 | 15                   | 15    |
| GPX     | D8                 | 27                   |       |
| Mosi    | D11                | 2                    | 23    |
| Miso    | D12                | 3                    | 19    |
| SCK     | D13                | 4                    | 18    |
| SS      | D10                | 1                    | 5     |
|         | GND                | 11, 16               | GND   |
|         | 3v3                | 9                    | 3v3   |
